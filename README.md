# Synapse AI

System wspierający spotkania biznesowe i inspekcje budowlane poprzez inteligentną analizę mediów (audio/wideo) i automatyzację procesów.

## Struktura Projektu
- `/frontend`: Aplikacja mobilna React Native (Expo).
- `/backend`: Serwer FastAPI obsługujący logikę LLM.
- `/docs`: Dokumentacja systemowa i Manifest Projektu.

## Szybki Start (Wkrótce)
Projekt jest w fazie inicjalizacji. Szczegółowe instrukcje zostaną dodane po wdrożeniu bazowych komponentów.
